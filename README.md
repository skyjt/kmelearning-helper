# KME 学习助手（kmelearning-helper）

> 一个给 `pc.kmelearning.com` 学习页用的 Chrome 扩展（Manifest V3）。
> 它会在页面右下角放一个“学习助手”浮窗，帮你**按目录顺序自动播放课程**、等平台确认完成，然后接着学下一门没学完的课。

---

## 这是什么

进入学习目录后，平时你要做的事是：点开一门课 → 等它播完 → 返回列表 → 再点下一门……重复几十遍。

这个扩展把这套重复操作自动化：

```
开始自动学习
   │
   ├─▶ 进入第一门未完成课程
   │      └─ 自动播放（锁定 1x），等平台显示“完成”
   │
   ├─▶ 返回上一级课程列表
   │
   └─▶ 继续下一门未完成课程 …… 直到全部学完
```

**它做的：** 减少重复点击，按平台自己的规则老老实实地播完每门课。
**它不做的：** 不倍速、不跳过、不伪造完成状态，也不替你答题或提交。

## 为什么默认不倍速、不跳过

`pc.kmelearning.com` 判断一门课是否完成，**不只看视频有没有播到结尾**，而是综合了心跳、真实学习时长、播放进度和完成状态。

所以：

- ❌ 第三方倍速插件就算把视频播完，也可能因为真实学习时长不够而不算完成。
- ❌ 直接拖进度条到结尾、或跳过播放，通常过不了平台的完成判断。
- ✅ 本扩展默认锁定 **1x**，平台没确认完成时就按 1x 补学 / 重播，稳稳通过。

## 快速开始

### 1. 安装扩展

**方式一：下载 ZIP（推荐新手）**

1. 打开 GitHub 仓库页 → 点 `Code` → `Download ZIP`，下载后解压。
2. Chrome 地址栏进入 `chrome://extensions`。
3. 打开右上角的 **开发者模式**。
4. 点 **加载已解压的扩展程序**，选中解压后的文件夹（里面要有 `manifest.json`）。

**方式二：Git 克隆**

```bash
git clone https://github.com/skyjt/kmelearning-helper.git
```

然后同样在 `chrome://extensions` 里 **加载已解压的扩展程序**，选中克隆下来的目录。

### 2. 开始学习

1. 登录并进入 `pc.kmelearning.com` 的学习目录页。
2. 确认页面右下角出现 **“学习助手”** 浮窗。
3. 点 **开始自动学习**。
4. 保持标签页打开就行，扩展会自动按“播完 → 返回 → 下一门”的顺序往下学。

> 💡 浮窗可以最小化成右下角的小图标，点一下或鼠标移上去就能展开。

## 功能一览

- 🪟 在学习目录页显示“学习助手”浮窗，并实时显示当前目录的**总进度**。
- ▶️ 点“开始自动学习”后，自动进入第一门未完成课程并播放。
- 📄 视频会自动播放；文档 / 材料类页面会保持活跃并等待完成标记。
- 🔁 一门课内容学完后，自动返回上一级课程列表，继续下一门。
- ⏱️ 返回前会检查课程记录里的“学习总时长”，不够课程要求学时就按 1x 继续补学。
- ⏭️ 自动跳过考试、测验、问卷、作业等**需要人工完成**的页面（不代答、不提交）。
- 🛡️ 默认抵消第三方倍速带来的“播完却不算完成”问题。

## 设置项

浮窗里可以开关这些行为：

| 设置 | 作用 |
| --- | --- |
| **自动播放** | 进入课程后自动开始播放视频 |
| **锁定 1x 真实学习** | 强制 1x 速度，确保积累真实学习时长 |
| **未完成自动补学** | 视频播完但平台没确认时，自动重播补足 |
| **总时长达标再返回** | 返回列表前检查学习总时长是否达到课程要求 |
| **跳过做题页** | 遇到考试 / 测验 / 问卷 / 作业自动跳过 |

## 常见问题

**浮窗没出现？**
确认你在 `pc.kmelearning.com` 的学习页，并刷新一次页面。第一次安装后已打开的页面需要刷新才会注入。

**学了一会儿停住了？**
平台页面结构更新可能导致识别失效。先刷新页面重试；仍有问题再更新扩展代码。

**能配合倍速插件用吗？**
不建议。平台会校验真实学习时长和心跳，倍速播完往往不算完成，反而拖慢进度。

## 开发

```bash
npm install        # 安装依赖
npm test           # 运行 Playwright 烟雾测试
npm run icons      # 重新生成图标
```

打包本地安装用的 ZIP：

```bash
mkdir -p dist
zip -r "dist/kmelearning-helper-v$(node -p "require('./manifest.json').version").zip" \
  manifest.json content.js styles.css icons README.md INSTALL.md LICENSE
```

### 文件结构

| 路径 | 说明 |
| --- | --- |
| `manifest.json` | Chrome MV3 扩展配置 |
| `content.js` | 注入到学习页面的核心逻辑 |
| `styles.css` | 浮窗和控件样式 |
| `icons/` | 扩展图标 |
| `tests/extension-smoke.mjs` | 基于 Playwright 的本地烟雾测试 |
| `tools/generate-icons.mjs` | 图标生成脚本 |
