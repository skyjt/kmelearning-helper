# 安装说明

> 这是 KME 学习助手的安装与上手指南。想了解扩展功能和原理，请看 [README](README.md)。

## 1. 获取项目

任选一种方式：

**方式一：下载 ZIP（推荐新手）**

在 GitHub 仓库页点 `Code` → `Download ZIP`，下载后解压。

**方式二：Git 克隆**

```bash
git clone https://github.com/skyjt/kmelearning-helper.git
```

## 2. 在 Chrome 中加载

1. 地址栏进入 `chrome://extensions`。
2. 打开右上角的 **开发者模式**。
3. 点 **加载已解压的扩展程序**。
4. 选中项目目录（里面要有 `manifest.json`）。
5. 打开或刷新 `https://pc.kmelearning.com/` 学习页面。

✅ 安装成功后，页面右下角会出现 **“学习助手”** 浮窗。

## 3. 开始学习

1. 登录平台并进入学习目录页。
2. 点浮窗里的 **开始自动学习**。
3. 保持标签页打开，扩展会自动进入未完成课程、按 1x 播放并等待平台完成标记。
4. 一门课学完后，自动返回上一级列表并继续下一门未完成课程。

> 💡 想调整行为（自动播放、跳过做题页等），点浮窗右上角的 **⚙**，卡片会翻转到背面的设置面，改完点「完成」翻回即可。

## 注意事项

- ⚠️ **不建议同时用第三方倍速插件**。平台会校验真实学习时长和心跳，倍速播完可能不算完成。
- ⏭️ 遇到考试、测验、问卷、作业等页面会**自动跳过**，扩展不会代答或提交。
- 🔄 如果页面结构更新导致扩展失效，先**刷新页面**重试；仍有问题再更新扩展代码。
