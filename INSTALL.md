# kmelearning-helper 安装说明

## 下载

可以任选一种方式获取项目：

- 在 GitHub 页面点击 `Code` -> `Download ZIP`，下载后解压。
- 或使用命令行克隆：

```bash
git clone https://github.com/skyjt/kmelearning-helper.git
```

## Chrome 安装

1. 打开 Chrome。
2. 进入 `chrome://extensions`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目目录，也就是包含 `manifest.json` 的文件夹。
6. 打开或刷新 `https://pc.kmelearning.com/` 学习页面。

安装成功后，页面右下角会出现“学习助手”浮窗。

## 使用

1. 人工登录平台并进入学习目录页。
2. 点击浮窗里的“开始自动学习”。
3. 保持标签页打开，扩展会自动进入未完成课程、按 1x 播放并等待平台完成标记。
4. 当前课程完成后，扩展会返回上一级课程列表并继续下一个未完成课程。

## 注意事项

- 不建议同时使用第三方倍速插件。平台会校验真实学习时长和心跳，倍速播完可能不会显示学习完成。
- 遇到考试、测验、问卷、作业等页面时，本扩展会跳过，不会代替用户作答或提交。
- 如果页面结构更新导致扩展失效，可以先刷新页面；仍有问题时再更新扩展代码。
